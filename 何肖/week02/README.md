# 第1周作业 - 5分类任务模型
学员姓名：何肖

## 项目说明
本项目实现了一个基于 PyTorch 的 5 分类神经网络模型。
任务目标：输入一个 5 维随机向量，模型判断哪一维数值最大，输出对应的类别（0~4）。

## 模型结构
- 模型类型：单层线性神经网络
- 输入维度：5
- 输出维度：5（对应 5 分类）
- 损失函数：交叉熵损失（CrossEntropyLoss）
- 优化器：Adam

## 主要功能
1. 随机生成数据集
2. 模型训练与评估
3. 准确率与损失曲线可视化
4. 模型保存与加载预测

## 运行结果
- 训练轮数：20 轮
- 训练样本：10000 个
- 模型最终准确率：接近 98%
- 输出包含：训练日志、模型权重、预测结果

## 文件说明
- `week01/作业1.py`：主程序代码（模型定义、训练、评估、预测）
- `README.md`：作业说明文档

## 使用方法
1. 运行主程序，自动开始训练
2. 训练完成后会显示 acc/loss 曲线
3. 自动加载模型并对给定测试向量进行预测
4. 输出每一条测试数据的预测类别
